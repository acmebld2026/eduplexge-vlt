9. 견적 (내부 계산 — 가격정책_참조표.md 기준)

  • 상담관리: "월 3회 이상" 권장 — 등원형태 표(주3~5회)와 정확히 매핑되지 않음. 첫 상담에서 등원 요일수를 명확히 확정할 것 (주3회=530,000 / 주4회=590,000 / 주5회=650,000 중 택1이 표준 구조).
  • 개별지도(검토안): 수학 주5회(공수2, 4주 환산 20콤마) + 중학수학 주3회(4주 환산 12콤마) = 32콤마
  • 고등 단가 33,000원 기준: 32 × 33,000 = 1,056,000원
  • ※ 이는 검토안 그대로의 상한 계산이며, 실제로는 소화도에 따라 콤마 수를 조정 (예: 절반 수준인 16콤마 = 528,000원부터 시작 검토 권장)
  • 제휴컨텐츠 10,000원 + 셀프리더 15,000원(최초 1회)

예상 초기월 합계 (주4회 상담관리 + 개별지도 16콤마 축소 시작 가정): 590,000 + 528,000 + 10,000 + 15,000 ≈ 1,143,000원 예상 초기월 합계 (주4회 상담관리 + 개별지도 32콤마 전체 검토안 가정): 590,000 + 1,056,000 + 10,000 + 15,000 ≈ 1,671,000원

→ 개별지도 물량이 견적을 크게 좌우하므로, 등록 상담 시 수학 투트랙의 실제 필요 콤마 수를 먼저 확정하고 가정용 요약 견적을 학부모께 안내할 것.
