2. 핵심 진단

  • 지연이는 공부량이 적거나 이해력이 낮은 학생이 아니다. 회독을 5~6번 하고 혼자 공부하려는 힘도 있으나, 과목별 병목과 시간 배분을 진단하지 않은 채 주어진 일정을 성실히 수행해 노력 대비 성적이 낮다.
  • 학습자립심 +30, 주도성 +40, CHAMP 이해 +30·사고 +21·문제해결 +27은 성장 자산이다. 반면 자기서사 -40과 낮은 호기심은 첫 고교 시험의 실패를 자기 가능성 축소로 해석하게 만든다.
  • 실제 희망은 경희대 통계학과에 가깝지만 상담 전에는 실패를 피하려고 목표를 "인서울"로 낮춰 말했다. 목표를 숨기지 않고 시험지·시간표·주간 행동으로 번역하는 상담이 필요하다.
  • 매니저는 "더 열심히"를 요구하기보다 시험지 정답률, 회독별 소요 시간, 국어·한국사 개념 공백을 측정해 공부 방법을 바꾸도록 해야 한다.

5. 상담·관계 전략

구분 전략
초기 라포 꼼꼼함과 성실성을 먼저 인정하고, 비효율을 능력 부족이 아니라 구조 문제로 설명
질문 방식 "몇 번 했니"보다 "1회독과 5회독에서 무엇이 달라졌니"처럼 결과가 남는 질문 사용
목표 상담 경희대 통계학과를 숨기지 않게 하고 필요한 등급·과목·주간 행동으로 분해
부모 역할 공부를 직접 통제하기보다 엔콕·학원·에듀플렉스 일정이 충돌하지 않도록 구조를 정리
매니저 배정 언어 영역 진단이 강하고, 학생의 조용한 수용을 동의로 오해하지 않는 매니저가 적합

구분 방향
등록 형태 상담&관리 우선. 개별지도는 초기부터 넣지 않고 국어·한국사 병목 관찰 후 처방적으로 추가
방학 등원 주 5일, 오전 8시 등원을 우선안으로 검토. 엔콕 일정과 실제 시작일을 7/14 세팅상담에서 확정
첫 2주 전체 시간표 작성, 시험지 분석, 국어·한국사 기반 진단, 회독별 목표와 완료 기준 설정
관리 루틴 START CHECK → SELF STUDY → DAILY CHECK, 회독 수보다 오답·어휘·개념 인출 기록 확인
부모 공유 주간 완료율·과목별 시간·병목 변화 중심으로 보고. 단순 체류 시간만 보고하지 않음
