# XCM16 공식 타입 기준표

타입 번호의 명칭은 기억이나 유사 표현으로 추정하지 않고 아래 표를 사용한다. 메인 타입과 히든 타입은 학생 원본 VLT 결과지와 녹취에서 각각 확인한다.

| 번호 | 공식 명칭 |
|---:|---|
| 1 | 엄친아 |
| 2 | 모범생 |
| 3 | 학원키즈 |
| 4 | 꼭두각시 |
| 5 | 협상이 |
| 6 | 허영이 |
| 7 | 아웃사이더 |
| 8 | 고집쟁이 |
| 9 | 유리구슬 |
| 10 | 오지랖 |
| 11 | 성실이 |
| 12 | 학습실지킴이 |
| 13 | 예민이 |
| 14 | 마음 콩밭 |
| 15 | 마니아 |
| 16 | 무심이 |

학부모용·매니저용 결과지에는 각각 `메인 TYPE N. 공식 명칭`, `히든 TYPE N. 공식 명칭` 형식으로 표시하고 두 결과물의 표기가 일치하는지 배포 전에 확인한다.
