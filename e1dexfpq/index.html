거주지 기준은 종암점이 가깝지만, 서원이의 기존 학원(수학)이 길음 인근이라 동선상 길음 상담을 받음. 어머님은 종암점 상담을 아직 받아본 적이 없고, 등원 결정 전에 종암점도 방문해 비교하고 싶어함. 원장도 "동선을 무시할 순 없다, 종암도 한번 가서 느낌을 보시라"고 직접 권유함 — 경쟁 지점 비교가 사실상 예정된 상태. 비용·프로그램 구성은 두 지점이 동일. 재상담/후속 연락 시 종암점으로 이탈했는지 여부를 반드시 확인할 것.

주의 어머니-자녀 소통 구조 (결과지 미기재)

ISTJ 어머니 × ESFP(P매우강) 아들 — 원장이 상담 중 "아이를 받아들이기 힘드실 것 같다"고 직접 진단한 조합. 계획·원칙 중심의 어머니와 즉흥적·감각적인 아들 사이에서 어머니가 채점·계획을 대신 해주는 밀착 개입이 이미 진행 중(아래 4번 참조). 등록 시 상담 전략에서 "부모 개입 축소 + 본인 책임 이전"을 핵심으로 다뤄야 함.

5. 수학 성적 하락 배경 (결과지 요약보다 상세)

1학기 반 배정에서 서원이 혼자 남학생, 나머지 전원 여학생인 반에 배정됨 → 담당 교사 교체 후 질문에 대한 피드백 방식에 부담을 느껴 질문 횟수 감소 → 본인 요청으로 친구가 있는 하위 레벨 반으로 이동 → 그 반에서 다른 남학생과 대인관계 트러블 발생 → 해당 친구가 퇴원하며 서원이도 다시 원래 레벨로 복귀 → 이 과정에서 학원 출석이 불안정해지며 기말 성적 급락. 원장 판단: 학원의 문제가 아니라 사회적 트러블에서 시작된 관리 공백. 수학 자체는 개념 비중이 낮아 중학교 때 빨리 잡아주면 되는 병목.

9. 견적 (내부 계산 — 가격정책_참조표.md 기준)

  • 등원형태 미정 — 오늘은 프로그램 세부 안내 전 단계(원장이 "동선부터 고민해보시라"며 상세 프로그램 설명은 보류함)
  • 확정 견적 대신, 실제 상담 중 학부모에게 언급된 구체 금액은 수학 개별지도뿐:
  • 수학 단과 주3회 × 12회/월 × 중등 30,000원 = 360,000원/월
  • 일반 등원형태(길음점 표준가, 가격정책_참조표.md 기준) 참고:
  • 주3회 상담+관리 = 530,000원 / 주4회 = 590,000원 / 주5회 = 650,000원
  • 상담만(1·2·3타입 대상, 서원이는 4타입이라 해당 안 될 가능성 높음) = 350,000원
  • 예상 시작 구성 (참고용, 미확정): 주3회 상담관리(530,000) + 수학 개별지도 주3회(360,000) + 제휴컨텐츠(10,000) + 셀프리더(15,000, 최초1회) ≈ 915,000원/월
  • ※ 학원 정리 범위(결정카드 2)에 따라 개별지도 콤마 수가 달라지므로 확정 견적 아님
